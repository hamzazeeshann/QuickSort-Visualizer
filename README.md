This is a QuickSort Visualizer that lets you understand how QuickSort algorithm thinks. You can tweak the settings to learn the algorithm as you like.
 - You can use Random Pivot or a Perfect Pivot (Median of Medians).
 - You can play it step by step or play it like a simulation.
 - Control the speed of the simulation.
 - Add your custom array.
 - See how it works on different cases such as random, sorted, reverse, nearly sorted
 - and more.

*How to Run:*
you just need to run the html file, everything is handled in it.
